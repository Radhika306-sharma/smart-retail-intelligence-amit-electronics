# Smart Retail Intelligence Platform

An AI-powered retail intelligence platform built for Amit Electronics, a 37-year-old electronics and car accessories business in Gwalior.

## Problem Statement
Small and medium retailers often rely on manual inventory tracking and intuition-based stocking decisions. This project aims to digitize retail operations and generate actionable business insights using sales data and machine learning.

## Objectives
- Digitize inventory management
- Analyze sales performance
- Generate AI-powered business insights
- Forecast future demand
- Identify top-selling and slow-moving products

## Features

### Public Website
- Product Catalog
- Category-wise Browsing
- WhatsApp Inquiry Integration

### Admin Dashboard
- Inventory Management
- Excel Upload
- Sales Analytics
- Low Stock Alerts
- AI Insights

## Tech Stack
- Next.js
- Supabase
- Python
- Pandas
- Scikit-Learn
- Vercel

## Dataset
Real sales and inventory data from Amit Electronics.
